everything you need to ship games faster

3d model generation

generate game-ready 3d models from text descriptions or reference images. supports Meshy, Tripo3D, and Rodin providers with direct Roblox upload.

sound effects

describe any sound effect and get multiple AI-generated variations instantly. adjust duration, influence, and download as mp3.

voice generation

generate character dialogue with batch processing. clone custom voices from audio samples for unique NPC personalities.

documentation editor

browse and edit your project's Luau source files with markdown rendering, syntax highlighting, and live split-view editing.

project board

kanban-style task management with drag-and-drop columns, priority labels, assignees, and due dates to keep your team organized.

model converter

convert 3d model files between formats for Roblox compatibility. drag-and-drop GLB, GLTF, FBX, or OBJ and export in the format you need.

built by gators, for game developers

RobloxDashboard was created by a team of University of Florida students who wanted to make Roblox game development faster and more accessible. by bringing AI-powered tools into a single, clean interface, we help developers spend less time on asset creation and more time on gameplay.
